# Racha Fácil

🔗 **Acesse o projeto:**
https://enilsonsn.github.io/racha-facil/

## Sobre

Projeto acadêmico desenvolvido por **Enilson Sousa Nascimento** com o objetivo de aplicar conceitos de desenvolvimento web utilizando HTML e CSS.

O Racha Fácil é uma interface para auxiliar na divisão de despesas entre grupos de pessoas de forma simples e intuitiva.

## Tecnologias

* HTML5
* CSS3

## Como executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/enilsonSN/racha-facil.git
   ```

2. Abra o arquivo `index.html` em seu navegador.
